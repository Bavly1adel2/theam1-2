# NeuroAI — HTML Template (Vite + Tailwind + Alpine)

NeuroAI is a modern AI SaaS + Tools Directory + Admin Dashboard **static HTML template**.

## Requirements
- Node.js 18+ recommended

## Install
```bash
npm install
```

## Dev
```bash
npm run dev
```

## Build
```bash
npm run build
```

Build output is written to `dist/`.
